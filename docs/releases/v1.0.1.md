这是 JD-Ray Local / 岗位黑话解码的首个公开体验版本。

## 主要能力

- 上传、拖拽或粘贴岗位 JD 截图。
- 可选同步 Markdown / TXT 简历参与匹配。
- 支持 Gemini 2.5 Flash 与 Qwen3 VL Plus。
- API Key、最近报告只保存在本机。
- 不保存原始岗位截图和简历全文。
- 输出岗位结论、黑话翻译、风险亮点、薪资逻辑、一天实况、最终建议、简历匹配和五维评分。
- 支持删除本机历史报告。
- 支持支付宝自愿打赏入口。

## 已知限制

- 没有远端后端服务，模型调用由当前设备直接请求模型服务商。
- 首次使用需要用户自行配置 Gemini 或 Qwen API Key。
- 中国区公开大规模分发前，请自行评估 APP 备案、隐私政策、模型服务合规和应用市场审核要求。

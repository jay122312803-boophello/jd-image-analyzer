# Changelog

## v1.0.1 - 2026-04-30

首个适合公开体验的 Android 本地版。

- 更新 Android 桌面图标与中文应用名：`岗位黑话解码`。
- 支持 Gemini 2.5 Flash 与 Qwen3 VL Plus。
- 支持上传、拖拽、粘贴岗位截图。
- 支持可选 Markdown / TXT 简历参与岗位匹配。
- 使用本机 `localStorage` 保存 API Key 和最近 20 条报告。
- 历史报告支持删除，不保存原始截图和简历全文。
- 报告页提供结构化岗位分析：结论、黑话翻译、风险与亮点、薪资逻辑、一天实况、最终建议、简历匹配和五维评分。
- 增加支付宝自愿支持入口。
- 提供已签名 Android release APK。

## v1.0.0 - 2026-04-29

- 完成 JD-Ray Local 的 React + Capacitor Android 初版。
- 完成前端直连模型 API、本地历史、移动端暗色 UI 和 Android 打包流程。
